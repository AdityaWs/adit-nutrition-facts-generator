# Adit's Nutrition Facts Generator

This is **Adit's Nutrition Facts Generator**, a simple webapp that was created for the purpose of creating nutrition facts label, you can use this app for free and it was originally created for helping me designing 3D product packaging which require a nutrition facts label design and placement on the product. However you can use the tool as you wish

To start generating a nutrition facts label for your product just visit [https://adit.madbyad.my.id/adit-nutrition-facts-generator](https://adit.madbyad.my.id/adit-nutrition-facts-generator) and start inputting your product nutrition information
